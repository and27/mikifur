# Mikifur

[![Project Status](https://img.shields.io/badge/Project%20Status-Development-yellow)](https://shields.io/)

![Mikifur Screenshot](public/screenshot.png)

Mikifur is an e-commerce store built with Next.js dedicated to selling pet apparel, specifically t-shirts.

## Key Features

*   **Product Catalog:** Browse a variety of pet t-shirts.
*   **Product Details:** View detailed information about each product, including descriptions, sizes, and images.
*   **Shopping Cart (In Development):** Add items to a shopping cart for purchase. 
*   **Checkout (In Development):** Secure checkout process. 
*   **Responsive Design:** Optimized for various screen sizes.

## Technologies Used

*   Next.js
*   React
*   Framer Motion
*   Tailwind CSS
*   TypeScript

## Installation

To run this project locally, follow these steps:

1.  Clone the repository:

    ```bash
    git clone [ttps://github.com/and27/mikifur]
    ```

2.  Navigate to the project directory:

    ```bash
    cd mikifur
    ```

3.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

To start the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
